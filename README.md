### How To Run
Clone the project in your local environment:
- HTTPS
  ```bash
  git clone https://github.com/pestana-catumbela/pokemonapp-2.git
  ```
- SSH
  ```bash
  git clone git@github.com:pestana-catumbela/pokemonapp-2.git
  ```

Into the project cloned directory, install all dependencies you need to run the project locally:
```bash
npm install
```

After that, run the bellow command to run the project:
```bash
npx expo start
```

It gives a QR Code, install Expo Go in your device and scan the code or install any amulator (android or ios).
